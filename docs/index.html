Awesome Features

Constellations

This application support GPS(L1/L5), Glonass, QZSS(L1/L5), BeiDou, Galileo(E1/E5) and NavIC(L5) constellations.

Powerful

It can support almost android GNSS API. You can do most of all functional test (including SUPL MSA, MSB test).

Customizability

In professional mode, you can freely combine the test tasks you want. Push the test config and do the custom testing.

NMEA Record

It can record the NMEA message to sdcard. We also provide the nmea analysis tools, You can do analysis easily.

Raw Data Record

It can record RAW measurements (need GNSS chip support) to sdcard. This file can be opened by Google GNSS Analysis app.

CEP Calculation

Circular Error Probability will auto calculate, The CEP result will display in time, and can store all the result to sdcard.
